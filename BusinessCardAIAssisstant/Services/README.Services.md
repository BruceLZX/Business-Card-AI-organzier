# Services

[中文说明 / Chinese](README.Services.zh.md)

Domain services for capture, vision/OCR parsing, AI enrichment, translation, and search.

Key services:
- `CaptureService`
- `OCRService` and `OCRExtractionService` (vision-first parsing with OCR fallback)
- `EnrichmentService` (web search + summaries + tags, create-time tags)
- `TranslationService`
- `SearchService`
